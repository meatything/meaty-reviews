Meaty Reviews

Extract reviews from Chrome Web Store extension pages

Drag this button to your bookmarks bar:

Extract Reviews

How to Use

  1. Install: Drag the "Extract Reviews" button above to your bookmarks bar
  2. Navigate: Go to any Chrome Web Store extension page with reviews
  3. Extract: Click the bookmarklet in your bookmarks bar
  4. Wait: It will automatically click "Load More" to collect reviews (up to 250)
  5. Download: A JSON file will automatically download when complete
Note: Chrome extensions cannot script the Chrome Web Store for security reasons. This bookmarklet runs directly in the page context, bypassing that restriction.
